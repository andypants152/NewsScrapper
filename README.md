# NewsScrapper

Scrapping up the news from The Verge, and saving it to a MongoDB instance.

## Built With
* Javascript
* Node
* Express
* Express-handlebars
* Body-Parser
* Bootstrap
* jQuery
* MongoDB
* Mongoose
* Request
* Cheerio
